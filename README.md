# AI Travel Planner

AI Travel Planner is a web-based travel planning application developed using Streamlit and OpenWeather API.

## Features

- Weather Dashboard
- Budget Analysis
- Destination Maps
- Packing Checklist
- Travel Tips
- PDF Export
- Responsive UI

## Technologies Used

- Python
- Streamlit
- OpenWeather API
- Plotly
- Folium
- Pandas

## Installation

```bash
pip install -r requirements.txt
streamlit run main.py
